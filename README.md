# Bar Reservation System - DDD Kata

PHP version of https://github.com/paulinejamin/live-coding-agicap-artisan-developpeur

## Getting started on Unix or Mac OS X system

### Init

```shell
make up
```

### Developing

Here a few commands that can help you :

* Launch unit tests
```shell
make unit-tests
```

* Launch code sniffer
```
make code-sniffer
```
